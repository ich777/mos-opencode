# MOS OpenCode

MOS Plugin for OpenCode - AI-powered coding assistant.

---

## Overview

This repository contains the **MOS plugin implementation**, optional helper
functions, configuration files (such as `settings.json`) and wrapper scripts
to integrate OpenCode within MOS.

### Binary Source

- opencode: [https://github.com/anomalyco/opencode](https://github.com/anomalyco/opencode)

---

## Build & Automation

This repository includes a **GitHub Actions workflow** used to build and package
the plugin for MOS.

The build process is fully automated and produces artifacts that can be
installed through the MOS Hub.
